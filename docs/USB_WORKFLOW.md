# Flujo USB

1. En el teléfono, abrir LOCKTRAIL.
2. Tocar `Exportar CSV a Downloads/LockTrail`.
3. Conectar el teléfono a la PC.
4. Abrir el almacenamiento del dispositivo.
5. Copiar:

```text
Downloads/LockTrail/
```

6. Abrir `desktop/locktrail_studio.py`.
7. Importar la carpeta copiada.
8. Generar el mapa HTML.
