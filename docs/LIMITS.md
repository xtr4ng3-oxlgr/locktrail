# Límites

LOCKTRAIL no garantiza recuperación del teléfono.

Factores que pueden impedir registros:

- GPS apagado,
- permisos revocados,
- batería agotada,
- modo avión,
- dispositivo apagado,
- app desinstalada,
- restricciones del fabricante,
- falta de conexión,
- almacenamiento inaccesible.

Para recuperación en tiempo real, usar también las funciones oficiales del sistema y acudir a autoridades.
