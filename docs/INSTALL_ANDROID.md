# Instalación Android

## Requisitos

- Android Studio.
- Android 10 o superior recomendado.
- Kotlin / Gradle desde Android Studio.

## Pasos

1. Abrir Android Studio.
2. Abrir la carpeta `android/`.
3. Esperar sincronización de Gradle.
4. Conectar el teléfono por USB.
5. Activar modo desarrollador y depuración USB.
6. Ejecutar la app en el dispositivo.
7. Conceder permisos de ubicación y notificaciones.

LOCKTRAIL usa un servicio visible cuando registra ubicación.
