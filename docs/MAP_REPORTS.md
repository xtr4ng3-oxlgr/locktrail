# Mapas y reportes

LOCKTRAIL Studio genera un HTML local con:

- puntos,
- ruta,
- inicio,
- fin,
- distancia aproximada,
- tabla de coordenadas,
- fuente de cada registro.

El mapa usa Leaflet y OpenStreetMap. No requiere API key de Google.

También puede intentar importar datos propios de Google Takeout, si el usuario exportó su historial desde su propia cuenta.
