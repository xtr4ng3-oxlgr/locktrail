# Security Policy

LOCKTRAIL is a defensive owner-device tool.

It must remain:

- visible,
- consent-based,
- local-first,
- export-controlled by the device owner.

Do not add:

- stealth tracking,
- hidden foreground services,
- silent email exfiltration,
- hidden boot persistence,
- remote control,
- spyware behavior,
- notification bypass,
- permission deception.

Reports should be used only for owner-controlled recovery and lawful reporting.
