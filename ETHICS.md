# Ética de uso

LOCKTRAIL existe para propietarios de dispositivos.

Principios:

- consentimiento claro,
- visibilidad permanente,
- control del usuario,
- exportación local,
- uso con autoridades cuando corresponda,
- cero rastreo oculto.

Este proyecto no autoriza:

- instalación en dispositivos ajenos,
- rastreo de personas sin permiso,
- ocultar notificaciones,
- enviar ubicación en silencio,
- manipular el código para vigilancia,
- distribuir builds modificadas para abuso.

Si un repositorio, fork o build usa LOCKTRAIL para fines abusivos, ese uso no está avalado por xtr4ng3.
